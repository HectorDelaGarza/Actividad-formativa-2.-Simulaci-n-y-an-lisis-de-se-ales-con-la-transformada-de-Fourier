# Actividad-formativa-2.-Simulaci-n-y-an-lisis-de-se-ales-con-la-transformada-de-Fourier

Este proyecto muestra la simulación de señales en el dominio del tiempo y su transformación al dominio de la frecuencia usando Python.

## Señales analizadas
- Señal senoidal
- Pulso rectangular
- Función escalón
- Señal amortiuguada

## Herramientas usadas
- NumPy
- Matplotlib

## Objetivo
Visualizar y analizar el comportamiento de señales usando la Transformada de Fourier.
